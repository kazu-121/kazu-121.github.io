制作の流れ

準備はスマホ・PCどちらでも可能です。
お客様とのやり取りは基本的にメールにて行います。

[ 1. お申し込み ]

お問い合わせページよりお申し込みいただけます。 その後プランや価格についてなど詳細をご連絡いたします。
口頭による認識違いをなくすため、お電話でのお問い合わせは受け付けておりません。ご了承ください。

[ 2. 素材の用意 ]

お客様専用のURLをお送りいたします

写真のスキャンをお願いします

コストや価格を抑えるため、写真のデータ化(スキャン)はお客様ご自身で行っていただきます。ご自宅にスキャナーがなくても問題ございません。コンビニのコピー機でスキャンすることが可能です。その後ご自身で専用ＵＲＬにアップロードをお願いいたします。

コメントの入力

専用ＵＲＬに写真をアップロードした後で、コメントを入力していただきます。その他いくつかムービーに取り入れるために質問事項がございますので、そちらの回答のご入力もしていただきます。

詳細につきましてはメールにてご案内いたします

[ 3. ムービーの制作 ]

アップロードしていただいた写真やコメントの最終的な確認を終えて、ムービー制作ができる状態(記入もれのない不備のない状態)であることが確認できた翌営業日から制作作業を開始いたします。およし２週間から１ヶ月の制作期間後、確認用ムービーをインターネット上でご覧いただきます。

[ 4. 修正 ]

確認用ムービーの内容をご覧いただき、特にテロップの誤字などの間違いがないかのご確認をお願いいたします。その他気になる点がございましたらご相談ください。可能な限り修正・対応いたします。（曲の変更や大幅な構成変更は別途料金となります。ご了承ください）

[ 5. 指定口座へご入金 ]

修正作業が終わり最終ムービーに問題がなければ、ご請求金額をご連絡いたします。（大幅な修正など別途料金がかかる旨の通知が事前にない場合は、当初のお見積りから変更はありません）
なお送料はお客様負担で全国一律1,000円(税別)をいただいております。ご請求金額をご確認いただきましたら、指定口座へのご入金をおねがいいたします。（指定口座へのご入金以外は対応しておりません。予めご了承ください）

[ 6. DVD納品 ]

料金の振込確認後、完成したＤＶＤ１枚をお送りいたします。

事前に式場で再生確認をしてください。
その他気になる点がございましたら、お気軽にお申しつけください。
